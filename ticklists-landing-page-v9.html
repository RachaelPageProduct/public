How it works

Set up once, use every day

Create your routine

Name it and add sections — Morning, Mid-morning, Lunchtime, Afternoon, Evening, whatever fits. One routine can cover your whole day, grouped by phase.

Step 2

Add tasks with day tags

Add each task and tag it to the days it applies — every day, weekdays only, just Saturdays. Each gets a colour. Tasks that don't apply today are automatically hidden, so you only ever see what's relevant right now.
